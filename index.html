Infrastructure Sizing Tool

Last updated: 02/18/2026  •  Interactive tool

Use this interactive sizing tool to determine the optimal infrastructure configuration for your deployment. Configure your user count, workload profiles, graphics requirements, and storage preferences to receive tailored VM recommendations and estimated costs.

Note

This tool provides estimates based on general best practices and Azure VM sizing guidance. Actual requirements may vary based on your specific applications and usage patterns. We recommend validating with a proof-of-concept deployment before production rollout.

Workload profile

Select the workload profile that best matches your users' daily activities. This determines the CPU and memory allocated per user session.

Workload Type
Select the primary workload for your user base
Tip

For mixed workload environments, select the profile that matches your heaviest user group, then adjust user counts accordingly. You can create multiple host pools with different VM sizes to optimize cost.

Graphics requirements

If your users require GPU-accelerated rendering (such as CAD, 3D modeling, video editing, or rich data visualization), select the appropriate GPU tier below.

GPU Configuration
Select graphics acceleration requirements
Important

GPU-enabled VMs have significantly higher costs. For multi-session deployments with GPU, the number of users per VM is limited by GPU memory. Ensure you validate GPU requirements with a proof-of-concept before committing to production.

Important

Cost estimates are approximate and based on publicly available Azure pricing. Actual costs may vary based on reserved instances, enterprise agreements, hybrid benefit, and other factors. Use the Azure Pricing Calculator for precise quotes.

Next steps

Now that you have your sizing recommendations, consider the following next steps to proceed with your deployment:

  • Review the complete prerequisites for Azure Virtual Desktop
  • Deploy Azure Virtual Desktop using the recommended VM configuration
  • Use the Azure Pricing Calculator for detailed cost analysis with your specific licensing
  • Configure identity and access management for your session hosts
  • Set up FSLogix profile containers for optimal user experience
